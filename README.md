# TransportX〈トランスポート・クロス〉
Copyright (c) 2026 automatic9045

https://transportx.okaoka-depot.com

TransportXとは、ユーザーが自由にマップや車両を作成できる3DCGバスシミュレーターです。作りたいものは何でも作れる、拡張性の高さが特長です。
